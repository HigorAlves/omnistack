## Prerequisites

You will need to install this tools to be able to setup your `environment`:

- [Git](https://git-scm.com)
- [NVM](https://github.com/nvm-sh/nvm)
- [Yarn](https://yarnpkg.com)
- [Docker](http://docker.com)
- [AWS CLI](https://aws.amazon.com/cli/)
- [Webstorm](https://www.jetbrains.com)

> We recommend that you use Webstorm as your IDE but you can use VSCode too.

## Environment

### Git
First you will need to install `git` you can follow the steps on the official site to handle the instalation on your machine,
after you have it installed you will need to update one config if your system is *case sensitive*:

```bash
git config core.ignorecase true
```

### Node Version Manager
You can follow the steps on the official site, to install it, it's just some steps over your bash and it's done, them
you will need to install the `nodejs` at `v16.19.1` you can do that by typing:

```bash
nvm install 16.19.1
```

After the process finishes you have to set it to your default version:
```bash
nvm alias default 16
```

### Docker & AWS CLI & Webstorm

You can follow their official guide.
